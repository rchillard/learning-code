min() selects the smaller of two values

The min() function in CSS will select the smaller of two values. In this example, we're using a single width rule to regulate the size of the div containing this content. Here's the rule:

    width: min(500px, 50%);

This is equivalent to:

    width: 500px;
    max-width: 50%;

You can also directly embed math in the function:

    width: min(500px + 10%, 50%);

max() selects the larger of two values

The max() function in CSS will select the larger of two values. In this example, we're using thesame width rule except substituing min for max. Here's the rule:

    width: max(500px, 50%);

Resize the window to see how this div will try to take up 50% of the width until 50% is less than 500px, after which it will simply stay 500px wide.

clamp()

The clamp() function in CSS is more useful for controlling typography. It allows you to set a minimum value, an ideal value, and a maximum value. See this rule: 

    font-size: clamp(1rem, 1.5vw, 3rem);

1rem is the minimum, 1.5vw the ideal, and 3rem is the maximum font size.
